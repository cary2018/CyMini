常用语句对照

1.查询数据 SELECT * FROM cy_visit 查询所有数据

SELECT * FROM cy_visit WHERE id=1000 查询指定ID数据

2.删除数据 DELETE FROM cy_visit 删除所有数据

DELETE FROM cy_visit WHERE id=1000 删除指定的第几条数据

DELETE FROM cy_visit WHERE region LIKE '%深圳%' 删除region包含"深圳"的数据

3.修改数据 UPDATE cy_visit SET ip=192.1.1.11 将所有 ip 字段里的值修改成"192.1.1.11"

UPDATE cy_visit SET ip=192.1.1.11 WHERE id=1000 指定的第几条数据把 ip 字段里的值修改成"192.1.1.11"

4.替换内容 UPDATE cy_visit SET `region`=REPLACE(`region`, '原始字符串', '替换成其他字符串')

5.清空数据ID重新从1开始 TRUNCATE cy_visit

6.添加字段：alter table cy_visit add clientType varchar(30) not null default '' comment'客户类型' after clientInfo;

7.删除字段：alter table 表名 drop 字段名

8.修改字段名：alter table 表名 change 旧名(要修改字段名) 新名
